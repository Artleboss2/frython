## 🥐 यह क्या है?

**Frython** एक प्रोग्रामिंग भाषा है जो **पूरी तरह फ्रेंच में** लिखे Python कोड को मानक Python में ट्रांसपाइल करती है। यह एक हास्यपूर्ण प्रोजेक्ट है, लेकिन **पूरी तरह कार्यात्मक** है — आप वाकई फ्रेंच में प्रोग्राम कर सकते हैं!

> *"जब आप फ्रेंच में 3 गुना अधिक शब्दों से कर सकते हैं तो अंग्रेजी में प्रोग्राम क्यों करें?" — Victor Hugo, शायद*

## 🚀 त्वरित शुरुआत

```python
afficher("नमस्ते दुनिया! 🐓")

naam = "Marie"
aayu = 25

si aayu >= 18:
    afficher(f"स्वागत है, {naam}!")
sinon:
    afficher(f"नमस्ते, {naam}!")
```

```bash
python -m frython namaste.fy
```
