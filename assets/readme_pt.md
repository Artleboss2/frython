## 🥐 O que é isso?

**Frython** é uma linguagem de programação que transpila código Python escrito **inteiramente em francês** para Python padrão. É um projeto humorístico, mas **totalmente funcional** — você pode realmente programar em francês!

> *"Por que programar em inglês quando você pode fazer em francês com 3 vezes mais palavras?" — Victor Hugo, provavelmente*

---

## ⚡ Instalação

```bash
pip install frython
```

## 🚀 Início rápido

```python
afficher("Olá mundo! 🐓")

nome = "Marie"
idade = 25

si idade >= 18:
    afficher(f"Bem-vinda, {nome}!")
sinon:
    afficher(f"Olá, {nome}!")
```

```bash
python -m frython ola.fy
```
