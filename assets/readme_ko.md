## 🥐 이게 뭔가요?

**Frython**은 **완전히 프랑스어**로 작성된 Python 코드를 표준 Python으로 트랜스파일하는 프로그래밍 언어입니다. 유머러스한 프로젝트이지만 **완전히 작동합니다** — 실제로 프랑스어로 프로그래밍할 수 있습니다!

> *"3배 더 많은 단어로 프랑스어로 할 수 있는데 왜 영어로 프로그래밍하나요?" — Victor Hugo, 아마도*

## 🚀 빠른 시작

```python
afficher("안녕하세요 세계! 🐓")

ireum = "Marie"
nayi = 25

si nayi >= 18:
    afficher(f"환영합니다, {ireum}!")
sinon:
    afficher(f"안녕하세요, {ireum}!")
```

```bash
python -m frython annyeong.fy
```
