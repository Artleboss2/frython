## 🥐 Was ist das?

**Frython** ist eine Programmiersprache, die Python-Code, der **vollständig auf Französisch** geschrieben wurde, in Standard-Python transpiliert. Es ist ein humorvolles Projekt, aber **vollständig funktionsfähig** — du kannst wirklich auf Französisch programmieren!

> *"Warum auf Englisch programmieren, wenn man es auf Französisch mit 3-mal mehr Wörtern machen kann?" — Victor Hugo, wahrscheinlich*

## 🚀 Schnellstart

```python
afficher("Hallo Welt! 🐓")

name = "Marie"
alter = 25

si alter >= 18:
    afficher(f"Willkommen, {name}!")
sinon:
    afficher(f"Hallo, {name}!")
```

```bash
python -m frython hallo.fy
```
