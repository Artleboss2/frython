## 🥐 这是什么？

**Frython** 是一种编程语言，可以将**完全用法语**编写的 Python 代码转译为标准 Python。这是一个幽默项目，但**完全可用** — 你真的可以用法语编程！

> *"当你可以用法语写出3倍的单词时，为什么要用英语编程呢？" — 维克多·雨果，大概是这么说的*

pip install frython

## 🚀 快速开始

```python
afficher("你好，世界！🐓")

nom = "Marie"
age = 25

si age >= 18:
    afficher(f"欢迎，{nom}！")
sinon:
    afficher(f"你好，{nom}！")
```

```bash
python -m frython nihao.fy
```
