## 🥐 これは何？

**Frython** は、**完全にフランス語**で書かれた Python コードを標準的な Python にトランスパイルするプログラミング言語です。ユーモラスなプロジェクトですが、**完全に動作します** — 本当にフランス語でプログラムできます！

> *「3倍の単語でフランス語でできるのに、なぜ英語でプログラムするの？」— ビクトル・ユーゴー、たぶん*

## 🚀 クイックスタート

```python
afficher("こんにちは世界！🐓")

namae = "Marie"
nenrei = 25

si nenrei >= 18:
    afficher(f"ようこそ、{namae}！")
sinon:
    afficher(f"こんにちは、{namae}！")
```

```bash
python -m frython konnichiwa.fy
```

## 📖 リファレンス

| Frython | Python |

| `si` | `if` |
| `sinon` | `else` |
| `sinonsi` | `elif` |
