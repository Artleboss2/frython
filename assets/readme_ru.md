## 🥐 Что это такое?

**Frython** — это язык программирования, который транспилирует код Python, написанный **полностью на французском**, в стандартный Python. Это юмористический проект, но **полностью рабочий** — вы можете реально программировать на французском!

> *"Зачем программировать на английском, когда можно делать это на французском с в 3 раза большим количеством слов?" — Виктор Гюго, вероятно*

## 🚀 Быстрый старт

```python
afficher("Привет, мир! 🐓")

imya = "Marie"
vozrast = 25

si vozrast >= 18:
    afficher(f"Добро пожаловать, {imya}!")
sinon:
    afficher(f"Привет, {imya}!")
```

```bash
python -m frython privet.fy
```
