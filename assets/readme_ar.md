## 🥐 ما هذا؟

**Frython** هي لغة برمجة تحوّل كود Python المكتوب **بالكامل باللغة الفرنسية** إلى Python قياسي. إنه مشروع فكاهي، لكنه **يعمل بالكامل** — يمكنك فعلاً البرمجة بالفرنسية!

> *"لماذا تبرمج بالإنجليزية عندما يمكنك فعل ذلك بالفرنسية بثلاثة أضعاف الكلمات؟" — فيكتور هوغو، على الأرجح*

---

```bash
pip install frython
```

## 🚀 البداية السريعة

```python
afficher("مرحبا بالعالم! 🐓")

ism = "Marie"
umr = 25

si umr >= 18:
    afficher(f"مرحباً، {ism}!")
sinon:
    afficher(f"أهلاً، {ism}!")
```

```bash
python -m frython marhaba.fy
```

---

## 📖 المرجع

| Frython | Python |
|---------|--------|
| `si` | `if` |
| `sinon` | `else` |
| `sinonsi` | `elif` |
| `tantque` | `while` |
| `pour` | `for` |
| `déf` | `def` |
| `retourner` | `return` |
| `afficher()` | `print()` |
| `Vrai` | `True` |
| `Faux` | `False` |
| `Rien` | `None` |
