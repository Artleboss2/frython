## 🥐 ¿Qué es esto?

**Frython** es un lenguaje de programación que transpila código Python escrito **completamente en francés** a Python estándar. Es un proyecto humorístico, pero **completamente funcional** — ¡puedes programar de verdad en francés!

> *"¿Por qué programar en inglés cuando puedes hacerlo en francés con 3 veces más palabras?" — Victor Hugo, probablemente*

---

```bash
pip install frython
```

## 🚀 Inicio rápido

```python
afficher("¡Hola mundo! 🐓")

nombre = "Marie"
edad = 25

si edad >= 18:
    afficher(f"¡Bienvenida, {nombre}!")
sinon:
    afficher(f"Hola, {nombre}!")
```

```bash
python -m frython hola.fy
```
